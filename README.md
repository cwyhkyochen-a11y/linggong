# 灵工 — 集团数字化底座

以 AI 对话为主要交互界面的集团数字化平台。六个子系统 + 一套生产力工具，覆盖项目、会议、日报、评估、人资、AI 基础设施，所有操作通过 OpenClaw 对话完成。

## 定位

传统 OA 让人去学页面、找按钮、填表单。灵工反过来：让 AI 做中间人，用户说人话，AI 操作系统，人只看结果。

## 系统架构

```
六个子系统（基础设施）
├── AI Infra 平台 — Skill / Key / Model / 用量 / 安全
├── 组织与人员 — 部门树 / 主数据 / 人资画像
├── 项目档案 — 看板 / 任务 / Wiki
├── 会议管理 — AI 判断 / 纪要 / 待办转入
├── 日报平台 — 日报 / 周报 / 月报
└── 人才评估 — 模板 / 分发 / 回收 / 汇总

一套生产力工具（日常提效）
├── 数据分析 — Excel/CSV → 图表 + 报告
├── 调研助手 — 给主题 → 结构化调研报告
├── 文档产出 — 口述 → 成稿
├── PPT 生成 — 大纲/稿 → 幻灯片
├── 表格处理 — 描述需求 → 处理结果
└── 代码辅助 — 生成 / Review / 调试
```

## 核心理念

- **对话优先**：不建传统管理后台，所有操作走 OpenClaw Skill
- **数据互通**：六个系统通过事件驱动串联，数据自动流动
- **项目挂载**：所有产出（文档、分析、PPT）都挂到项目上
- **事实数据**：提供工作产出的事实数据，不做正式绩效评估
